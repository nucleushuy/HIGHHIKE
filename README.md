This is HighHike, a project for browsing through CA trails!

California is home to many different parks and trails, but hikers may struggle to find new routes that match their preferences. Our goal is to create an interactive page that provides a resource for trails around the state. Users can search, filter, and add onto the current trail list while getting recommendation trails based on specified criteria.
